⚠️ Configura una vez
Sigue estos pasos para conectar la app a tu Google Sheets. Solo se hace una vez.

🔧 Pasos de Configuración

1

Crear Google Sheets

Abre sheets.google.com, crea un nuevo documento y nómbralo MiFinca. Crea dos hojas: Inventario y Eventos.

2

Encabezados en Inventario

En la hoja Inventario, escribe en la fila 1:

A1: ID B1: Nombre C1: Tipo D1: Socio E1: Fecha F1: Obs
3

Encabezados en Eventos

En la hoja Eventos, escribe en la fila 1:

A1: Fecha B1: ID C1: Tipo D1: Obs
4

Abrir Apps Script

En el Sheets, ve a Extensiones → Apps Script. Borra todo el código que aparece y pega el siguiente:

const SHEET_ID = SpreadsheetApp.getActiveSpreadsheet().getId(); function doPost(e) { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.openById(SHEET_ID); const hoja = ss.getSheetByName(data.hoja); hoja.appendRow(data.fila); return ContentService.createTextOutput( JSON.stringify({ok: true}) ).setMimeType(ContentService.MimeType.JSON); } function doGet(e) { const ss = SpreadsheetApp.openById(SHEET_ID); const inv = ss.getSheetByName("Inventario") .getDataRange().getValues(); const evt = ss.getSheetByName("Eventos") .getDataRange().getValues(); return ContentService.createTextOutput( JSON.stringify({inventario: inv, eventos: evt}) ).setMimeType(ContentService.MimeType.JSON); }
5

Publicar como Web App

En Apps Script: Implementar → Nueva implementación → tipo: Aplicación web → Ejecutar como: Yo → Quién tiene acceso: Cualquier usuario → Implementar.

Copia la URL de la aplicación web que aparece.

6

Pegar URL aquí abajo

Pega la URL en el campo de abajo y presiona Guardar.

📊 Conectar Excel (PC)

En Excel en tu PC:
1. Datos → Obtener datos → Desde web
2. Pega la misma URL del Apps Script
3. Excel importará los datos como tabla
4. Usa Actualizar todo cada vez que quieras sincronizar
